# Movie Analytics Dashboard

An interactive Streamlit application for analyzing movie ratings, genres, user demographics, and viewing trends using data mining and machine learning techniques.

## Live Demo

🚀 https://movieanalytics-dashboard.streamlit.app/

## Features

- Genre analysis
- Rating distribution analysis
- User demographic insights
- Occupation and age-based preferences
- Interactive visualizations
- K-Means clustering

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly

## Learning Outcomes

- Exploratory Data Analysis (EDA)
- Data Visualization
- User Behavior Analysis
- Clustering with K-Means
